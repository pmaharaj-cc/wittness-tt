WiTTness. Protect our kids. (interactive demo)
==============================================

NO SERVER
  Static files only. On GitHub Pages you get a public link, but there is still
  no backend. Trips live in each browser and sync across tabs on that link.

OPEN LOCALLY
  Double-click:  open-demo.bat
  Or:           index.html

LIVE DEMO
  Home:     https://pmaharaj-cc.github.io/wittness-tt/
  Parent:   https://pmaharaj-cc.github.io/wittness-tt/?role=parent
  WiTTness: https://pmaharaj-cc.github.io/wittness-tt/?role=wittness
  Trip:     https://pmaharaj-cc.github.io/wittness-tt/?role=wittness&trip=ABC

SOLO TEST (one computer)
  1. Parent tab: create trip, START, END
  2. New tab: paste WiTTness link from parent screen
  3. Confirm delivery (open 3 tabs total for quorum of 3)

TWO PEOPLE
  Parent shares WiTTness link after END. Friend confirms on their phone.
  Note: different devices need the same deployed URL but will not share trips
  until a real backend exists. For cross-device today, use one shared laptop
  or wait for the production MVP.

See README.md in this folder for more detail.
