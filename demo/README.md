# WiTTness — Interactive demo

**Protect our kids.** Try one full trip as **Parent** or **WiTTness**.

## Local

Double-click `open-demo.bat` or `index.html`.

## After GitHub Pages deploy

| Link | Role |
|------|------|
| `https://YOU.github.io/wittness-tt/` | Home |
| `...?role=parent` | Parent |
| `...?role=wittness` | WiTTness |
| `...?role=wittness&trip=ABC` | WiTTness — specific trip |

## Solo test

1. Parent: create → START → END  
2. Open WiTTness link in new tab(s) → Confirm until quorum (3)  
3. Parent tab: Trip complete
